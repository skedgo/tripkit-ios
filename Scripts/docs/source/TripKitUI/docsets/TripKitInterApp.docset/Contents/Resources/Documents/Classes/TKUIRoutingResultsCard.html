  • init(destination:initialPosition:)

    Initializes and returns a newly allocated card showing results of routing from current location to a specified destination. The card will be placed at the specified initial position.

    If the initial card position is not provided to the initializer, the value specified in the global configuration will be used. The default position is .peaking.
